# 运行《臭气退退退》

如果 `http://127.0.0.1:5173` 出现 `spawn EPERM`，这是当前 Windows 环境阻止 Vite 开发服务器内部子进程导致的，不是游戏代码错误。

请使用下面方式运行稳定版：

```bash
cd /d D:\desktop\health
npm.cmd run dev
```

运行成功后打开：

```txt
http://127.0.0.1:5174
```

也可以直接双击项目根目录的：

```txt
start-game.cmd
```

看到命令窗口里显示“臭气退退退已启动”后，不要关闭这个窗口；关闭窗口网页服务也会停止。
