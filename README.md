# 《臭气退退退》健康传播科普小游戏

## 1. 项目简介

《臭气退退退》是一款健康传播主题的网页小游戏，聚焦“科学认识狐臭与正确除臭”。玩家需要根据不同角色的狐臭程度，选择合适的除臭工具，帮助角色从“尴尬出汗”状态逐渐恢复到“清爽开心”状态。

游戏希望用清新、卡通、轻松幽默的方式降低狐臭话题带来的尴尬感，让玩家在互动中了解：

- 狐臭的基本成因；
- 常见除臭误区；
- 科学的日常护理方式；
- 重度狐臭需要正规医学治疗。

> 重要原则：游戏不能羞辱、丑化有狐臭的人。所有角色都应该被友好、温和地呈现，游戏的核心是“帮助角色解决困扰”，而不是嘲笑角色。

### 5.3 游戏页 Game

游戏页需要包含：

1. 当前角色图片；
2. 当前角色名称和狐臭程度；
3. 气味值进度条；
4. 绿色气味云动画；
5. 正确工具栏；
6. 错误工具栏；
7. 科普小贴士弹窗；
8. 错误次数提示；
9. 返回人物选择按钮。

建议布局：

- 上方：角色名称、气味条、错误次数；
- 中间：角色人物图 + 气味云；
- 下方：工具卡片；
- 点击工具后：弹出科普卡片。

气味值越高，角色周围的绿色气味云越明显；气味值下降后，气味云逐渐减少；成功时显示清爽闪光、泡泡或微风动画。

---

### 5.4 结果页 Result

#### 成功状态

触发条件：气味值 `<= 0`。

显示内容：

- 标题：臭气退退退成功！
- 显示角色开心状态图片；
- 总结：科学除臭三步——规范清洁、保持干爽、必要时就医；
- 按钮：再玩一次；
- 按钮：重新选择角色。

#### 失败状态

触发条件：连续选择 3 次错误工具。

显示内容：

- 标题：别灰心，避开误区才是第一步！
- 显示角色尴尬状态图片；
- 文案：狐臭不能靠香水、偏方或过度清洁解决，要选择科学方法哦。
- 按钮：重新挑战；
- 按钮：返回人物选择。

失败页语气必须友好，不能批评玩家，也不能嘲笑角色。

## 11. 游戏机制

### 11.1 初始气味值

```js
mild: 40
medium: 70
heavy: 100
```

### 11.2 工具效果规则

1. 如果玩家选择了适合当前角色程度的正确工具：
   - 气味值减少完整 `effect`。

2. 如果玩家选择了正确工具，但不是当前角色最关键的方法：
   - 气味值减少 `effect / 2`。
   - 弹窗提示：这个方法有帮助，但不是当前最关键的方法。

3. 如果玩家选择错误工具：
   - `mistake + 1`；
   - 气味值按照 `effect` 增加或保持不变；
   - 工具卡片轻微抖动；
   - 弹出误区提示。

4. 如果连续错误次数达到 3：
   - 进入失败页。

5. 如果气味值小于或等于 0：
   - 进入成功页。

### 11.3 角色状态变化

- 人物选择页：使用 `正常` 图片；
- 游戏开始和气味值较高时：使用 `尴尬` 图片；
- 成功后：使用 `开心` 图片；
- 气味值下降到初始值一半以下时，可以让角色逐渐变轻松。

## 17. 课堂展示文案

《臭气退退退》是一款健康传播主题的卡通网页小游戏。游戏围绕“狐臭科普”展开，设置轻度、中度、重度三类角色，玩家需要根据不同程度选择合适的除臭工具。通过气味值变化、角色状态反馈和科普弹窗，游戏引导用户认识狐臭成因，避开香水掩盖、偏方涂抹、过度清洁等误区，理解规范清洁、止汗除臭、透气衣物、清淡饮食和正规医学治疗等科学应对方式。作品希望用轻松幽默的互动形式降低敏感健康话题的尴尬感，让健康知识更容易被理解和接受。
